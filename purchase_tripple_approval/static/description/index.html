Triple Approval on Purchase Order in Odoo

Purchase Approval Process

Approval Purchase Order Triple: Purchase Manager -> Finance Manager -> Director Approval.

System is flexible to have approval buttons based on amounts configured on company form. For example if the purchase amount is less than 5000 then it only requires single level approval but if the amount of purchase order is greater than 5000 and less 8000 then it will need Purchase manager + Finance manager approval.


Main Features


  • Added Finance Approval Start Range, Stop Range.
  • Added Director Approval Start Range, Stop Range.
